== Description ==

The DR Express plugin from Digital River allows you to easily integrate with Digital River's Global Commerce platform - leveraging its Cart, Checkout and Account features

== Installation ==

Follow these steps to setup DR Express

1. Upload plugin through the 'Plugins' menu in WordPress or via FTP
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure Global Commerce options
4. Import Products
