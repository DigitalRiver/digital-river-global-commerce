== Description ==

The Digital River Global Commerce plugin from Digital River allows you to easily integrate with Digital River's Global Commerce platform - leveraging its Cart, Checkout and Account features

== Installation ==

Follow these steps to setup Digital River Global Commerce

1. Upload plugin through the 'Plugins' menu in WordPress or via FTP
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure 'Digital River > Settings'
4. Import Products
